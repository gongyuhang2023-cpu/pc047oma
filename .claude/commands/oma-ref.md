# OMA Book Reference

Quick reference to OMA (Orchestrating Microbiome Analysis) chapters.

## Usage
- `/oma-ref` - Show chapter overview
- `/oma-ref diversity` - Chapters on diversity analysis
- `/oma-ref daa` - Chapters on differential abundance analysis
- `/oma-ref function` - Chapters on functional analysis

## Chapter Index

### Foundations (Ch 1-6)
- Chapter 1: Introduction to microbiome analysis
- Chapter 2: Data structures
- Chapter 3: Data import
- Chapter 4: Quality control
- Chapter 5: Data transformation
- Chapter 6: Normalization

### Diversity Analysis (Ch 7-10)
- **Chapter 7**: Alpha diversity - richness, evenness, Shannon, Simpson
- **Chapter 8**: Beta diversity - Bray-Curtis, UniFrac, ordination (PCoA, NMDS)
- **Chapter 9**: Statistical testing - PERMANOVA, ANOSIM, dispersion tests
- **Chapter 10**: Interpretation - combining alpha/beta patterns

### Differential Abundance (Ch 11-15)
- Chapter 11: DAA overview and challenges
- Chapter 12: Compositional data issues
- Chapter 13: Log-ratio approaches
- **Chapter 14**: ANCOM-BC methodology
- **Chapter 15**: ALDEx2, MaAsLin2 comparisons

### Functional Analysis (Ch 16-20)
- **Chapter 16**: Functional annotation overview
- **Chapter 17**: HUMAnN and pathway analysis
- Chapter 18: Gene family analysis
- Chapter 19: Species contribution
- Chapter 20: Multi-omics integration

### Advanced Topics (Ch 21-27)
- Chapter 21: Machine learning
- Chapter 22: Network analysis
- Chapter 23: Time series
- Chapter 24: Meta-analysis
- Chapter 25-27: Special topics
